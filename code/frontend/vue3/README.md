# base_web

基础前端模板

## 项目结构
src/
├── assets/ 静态资源
├── components/ 公共组件
├── pages/ 页面
├── router/ 路由
├── store/ 状态管理
├── App.vue 根组件
├── main.ts 入口文件


## 开发
```bash
# 实际执行的package.json脚本
# 安装依赖
pnpm i

# 启动开发环境
pnpm run dev

# 打包
pnpm run build
```
