## 系统架构设计

```
[客户端上传图片]
        ↓
[本地 FastAPI 服务] ←─ 4核8G CPU，纯Python
        ├─ 1. OCR 识别 → 得到 [文字+位置坐标] 
        ├─ 2. 拼接上下文 → 构造 LLM Prompt
        ├─ 3. 并发异步POST 请求 → 外部 LLM API (如: 通义千问 / GPT-4 / Claude)
        ├─ 4. 解析返回 → 得到替换后文本
        └─ 5. 图像重绘 → OpenCV/PIL 绘制新文字，输出 PNG/JPG
        ↓
[返回替换后图片]
```
