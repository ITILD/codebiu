
# 开发环境指南 (Development Environment Guide)

## **1. 概述**

本文档旨在帮助开发人员快速搭建本项目的标准开发环境，确保团队开发环境的一致性。项目基于 **FastAPI** 框架，使用 **uv** 进行 Python 依赖管理和虚拟环境管理。

---

## **2. 核心技术与工具**

| 组件             | 推荐版本/选择                                  | 说明                                                                 |
| :--------------- | :--------------------------------------------- | :------------------------------------------------------------------- |
| **编程语言**     | Python `3.12` 或更高                           | 本项目的主要开发语言。                                               |
| **Python版本和依赖管理**  | **uv** (推荐) 或 conda                          | 用于安装和管理 Python 解释器版本，以及创建虚拟环境。                 |
| **Web 框架**     | **FastAPI** `0.104+`                           | 现代、高性能的 Web 框架。                                            |
| **IDE/编辑器**   | **Visual Studio Code (VSCode)**                 | 推荐使用，并安装必要扩展。                                           |
| **版本控制**     | Git                                            |                                                                      |
| **代码托管**     | Gitea                                          | 项目的 Git 远程仓库。                                                |
| **数据库**       | **PostgreSQL** `17` 或更高                       | 主数据库。                                                           |
| **数据库管理工具** | **Navicat** `16` 或更高                       | 数据库管理工具。                                                     |
