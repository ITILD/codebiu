## 技术基础

### 1. 主要技术栈
- git 基础使用和多人版本管理
- python 后端基础语法和常用库的使用，项目上的代码规范
- uv/conda 后端开发虚拟环境管理 [uv 安装配置](https://uv.doczh.com/getting-started/installation/)
- fastapi 注意学习异步开发 [fastapi 官方文档](https://fastapi.tiangolo.com/)
-
- postgre sql 基础使用和数据库设计
- numpy、opencv、onnx 基础计算库和小模型相关
- langchain、langraph 了解基础LLM部署调用和复杂图结构
- pgvector 向量数据库的基础使用
-
- ts、vue 前端基础语法和常用库的使用，项目上的代码规范
- docker/docker compose 服务器端基础部署和快速开发环境构建
- nginx 基础反向代理

### 2. 常用工具

- vscode IDE基础使用和常用插件 [vscode下载](https://code.visualstudio.com/)
- navicat 数据库管理工具
- edge/chrome 浏览器
