## 课程内容
入职第一天，带你的人说："项目代码拉下来启动看一下，看好了，自己试试加一个小功能。"
### 具体内容
1. 部署基础环境,vscode 基础ide安装, postgre 数据库服务，navicat 数据库管理图形界面，git 代码版本管理，uv python版本和依赖库管理器
2. git 拉取 http://zwork1.w1.luyouxia.net/gitea/codebiu_2025 上的基础文档库和基础后端代码库
3. 在基础后端代码库使用uv sync安装当前代码库python依赖
4. 在基础代码库的src/app.py文件里添加config.dev.yaml配置文件，启动调试服务，成功后访问http://127.0.0.1:2001/template/docs查看模版代码
5. src下新建module_ai模块，参考template下内容，实现增加数据的功能，最后把整个模块在app.py中引用。

### 必备代码基础
1. 面向对象-类的定义_类和对象的关系！！！
2. Python 程序格式_缩进_行注释_段注释
3. 模块导入_import 和from_import语句详解和区别！！！

### 必备知识点
1. vscode的基础操作
2. fastapi框架学习
3. 基础的调试报错和定位错误能力
4. 代码规范！！！
