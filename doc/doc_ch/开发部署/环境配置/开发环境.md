# 开发环境配置

## 环境列表

| 软件名称 | 用途                      | 官网地址                                          | 说明                                               |
| -------- | ------------------------- | ------------------------------------------------- | -------------------------------------------------- |
| vscode   | IDE                       | [vscode](https://code.visualstudio.com/)          | 强大的代码编辑器，支持多种编程语言和丰富的扩展插件 |
| uv       | Python 包和虚拟环境管理器 | [uv](https://github.com/astral-sh/uv)             | 快速创建和管理 Python 项目环境，替代 pip 和 venv   |
| fnm      | Node.js 版本管理器        | [fnm](https://github.com/Schniz/fnm)              | 快速切换不同版本的 Node.js，比 nvm 更快            |
| pnpm     | JavaScript 包管理器       | [pnpm](https://pnpm.io/)                          | 比 npm 和 yarn 更快更节省磁盘空间的包管理器        |
| winget   | Windows 包管理器          | [winget](https://github.com/microsoft/winget-cli) | Windows 官方包管理器，用于安装和管理软件包         |
## 注意事项
win确保系统已经安装PowerShell 且权限设置对,以管理员身份打开PowerShell，然后执行
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

## 1. python 环境配置

#### 1.1 安装 uv 环境管理器
linux/macOS安装
```sh
curl -LsSf https://astral.sh/uv/install.sh | sh
```
win安装
```sh
winget install --id=astral-sh.uv  -e
```
win无winget
```sh
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"

```
#### 1.2 常用命令

```sh
# 项目管理
# 创建和开发带有 pyproject.toml 的 Python 项目。
uv -V # 查看版本
uv init # 创建新 Python 项目
uv add # 为项目添加依赖
uv remove # 从项目移除依赖
uv sync # 已有项目安装python环境和依赖
# 无法连接远程下载源 时，请使用
# uv add 你要下的的包 --default-index https://pypi.tuna.tsinghua.edu.cn/simple
```

## 2. nodejs 环境配置

### 2.1 配置 fnm nodejs pnpm

linux/macOS
```sh
# 安装fnm （nodejs管理器linux/macOS）
curl -fsSL https://fnm.vercel.app/install | bash
```
win
```sh
winget install Schniz.fnm 
```
win 无 winget
```sh
https://github.com/Schniz/fnm/releases 下载fnm压缩包->解压->配置环境变量
```
fnm使用
```sh
# 创建配置文件，可以在 PowerShell 中运行以下命令：
if (-not (Test-Path $profile)) { New-Item $profile -Force }
# 要编辑您的配置文件，可以在 PowerShell 中运行以下命令：
Invoke-Item $profile
# 配置文件末尾添加以下内容：
fnm env --use-on-cd --shell powershell | Out-String | Invoke-Expression
```
fnm使用
```sh
fnm -V # 查看版本
fnm list # 查看本地已安装 Node.js 版本
fnm list-remote # 查看全部可安装的版本
fnm install v22.19.0 # 安装 Node.js稳定版本
fnm use v22.19.0 # 使用稳定版本
fnm default v22.19.0 # 默认版本切换
fnm uninstall v22.19.0 # 卸载
```

安装pnpm
```sh
# 先fnm install v22.19.0 # 安装 Node.js稳定版本 看node环境完成
node -v
# 使用corepack（nodejs包管理器）
corepack prepare pnpm@latest --activate
# 使corepack生效
corepack enable
```

### 2.2 常用命令

```sh
pnpm install # 安装依赖
pnpm run dev # 启动项目
```
